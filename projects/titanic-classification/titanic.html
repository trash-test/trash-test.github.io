Kaggle's Titanic Classification Challenge

 Find the source code here.

Overview

A project to test my understanding after finishing Andrew Ng's Machine Learning course using Kaggle's dataset. The objective is to predict a passenger survivability during the Titanic disaster.

My accuracy on the test set is currently 0.78947.

What's in the source code?

There are 3 Jupyter notebook files: analysis, process and train. They do exactly as expected. analysis.ipynb contains my analysis of data, process.ipynb is where I process the data, and train.ipynb documents the training process.

What I did

After the analysis, my decision on the process steps is:

  1. Extract Title features from Name features. Change Mlle to Miss and Mme to Mrs since they are essentially the same.
  2. Fill in missing Age info based on Title, missing Fare info based on Pclass, and missing Embarked info by most common port.
  3. Reduce numbers of unique titles (combine titles with less than 10 examples). Convert Title, Sex and Embarked port data into numerical labels (or onehot encode them).
  4. Combine SibSp and Parch into one FamilySize feature.
  5. Norminalize Fare and Age.
  6. Drop Ticket, Name, PassengerId and Cabin columns.

Then the data is trained using multiple models: Logistic Regression, Support Vector Machine, Linear Support Vector Machine, K-Nearest Neighbor, Naive Bayes, Perceptron, Stochastic Gradient Descent, Random Forest and Decision Tree. In the end, Support Vector Machine does the best and was picked for the final prediction on test set.
