Graph Application with Java

 Find the source code here.

Overview

This is my final project for Data Structures class. This project follows the requirements of Prof. Howe detailed on his page. It is an implementation of graph data structure in Java, which is then used to create an application that allows user to build graphs interactively.

What can it do?

The users can:

  • Add node by left-click, remove node by right-click, move node by dragging.
  • Add edge by dragging between two nodes, remove edge by clicking two nodes the edge is connecting.
  • Change nodes content and display the length of edges (scaled down from the pixel length).
  • Find shortest path between two nodes, using Dijkstra algorithm, based on the length of the edges between them.
