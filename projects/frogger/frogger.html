Frogger in Python

 Find the source code here.

Overview

This is my final project for Intro to ComSci class. It is a simple recreation of the classic game Frogger using Zelle's graphics.py

library.

What I did

I wrote the code. The ingame sprites are editted by me from existing sprites. Link to source sprites can be found in the README.md of the source code.

Gameplay

The objective is to move the frog from one side of the street to the other without colliding with obstacles. The player controls the frog using arrow keys.
