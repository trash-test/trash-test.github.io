Discrete Fourier Transform Visualization in PyQt

 Find the source code here.

Overview

This is my personal project inspired by 3Blue1Brown's video on Fourier Transform and Heat Equation. This projects uses Discrete Fourier Transform, first breaking a given function into a finite set of frequencies, each is then represented by a circle rotating at a constant rate. These cirlces are stacked upon each other and the tip of last one in the sequence will draw out the shape of the given function. It can either take in an .svg file as the function or the user can click and drag on the screen to draw the function.

This project is done with PyQt for the graphics, since I thought I need to learn a decent Python GUI library.

Note

Ideally, the input .svg should only consist of one continuous path. Having multiple discontinuous paths will yield result with lines connecting those paths.

The input .svg file should be placed in the svg folder. You can also provide the program with a path to the .svg file in the source code.
